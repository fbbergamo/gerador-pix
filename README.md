# gerador-pix

Exemplo para geracao de QRCode PIX padrão BRCODE https://www.gerarpix.com.br/

## Setup

Instalar Nodejs, versão no `.tool-versions`

Servidor

```
npm install

npm start
```


Página estática

Webpack Watcher - Compilador de JS e CSS

```
cd website
npx webpack --mode development  --watch
```
